# VibeCode OS — Мастер доустановки (Minimal → Full)

## 📋 Описание

Если вы установили **VibeCode OS Minimal** (консольная версия) и хотите доустановить дополнительные компоненты, используйте мастер доустановки.

## 🚀 Быстрый старт

```bash
# Через Makefile (рекомендуется)
sudo make upgrade

# Или напрямую
sudo bash ./scripts/minimal-upgrade.sh
```

## 📦 Что можно установить

Мастер предлагает следующие категории компонентов:

### 1. Терминал и оболочка
- **Kitty** — GPU-ускоренный терминал
- **Zsh** + **Oh My Zsh** — оболочка с плагинами
- **Starship** — кроссплатформенный промпт
- **CLI-утилиты:**
  - `eza` — современная замена `ls`
  - `bat` — замена `cat` с подсветкой
  - `fd` — быстрая замена `find`
  - `ripgrep (rg)` — быстрый поиск текста
  - `fzf` — нечёткий поиск
  - `zoxide` — умные переходы по каталогам
  - `btop` — монитор ресурсов

### 2. Языки программирования
- **Python** (3.11, 3.12) через `pyenv`
- **Node.js** (LTS + latest) через `nvm`
- **Rust** через `rustup`
- **Go** (свежая версия)
- **Java 21 LTS** через `SDKMAN!`

### 3. Редакторы и IDE
- **VSCodium** с расширениями (GitLens, Docker, Python, темой Tokyo Night)
- **Neovim** + **AstroNvim** — готовая конфигурация
- **Zed** — быстрый современный редактор

### 4. Инструменты разработчика
- **Git** — контроль версий
- **lazygit** — TUI для Git
- **Docker** + **Docker Compose** — контейнеризация

### 5. AI-стек
- **Ollama** — локальные LLM
- **Open WebUI** — веб-интерфейс для моделей
- **ai-chat** — терминальный AI-чат
- **Aider** — AI-парное программирование
- **Python AI-библиотеки** (PyTorch, Transformers, LangChain, LlamaIndex)
- **ComfyUI** — генерация изображений

**⚠️ Требует много места:** ~10-20 ГБ для моделей и инструментов

### 6. Драйверы NVIDIA
- Проприетарные драйверы для видеокарт NVIDIA
- Автоматическое определение оборудования

### Команды меню

| Команда | Описание |

| `1-6` | Установить конкретный компонент |

| `R` | Перепроверить установленные компоненты |

### Автоматическая установка

Для автоматической установки всех компонентов:

```bash
# Запустите мастер и выберите "A" для установки всего
sudo make upgrade
```

## 💡 Советы

1. **Проверьте место на диске:**
   ```bash
   df -h
   ```

2. **Минимальные требования:**
   - **Терминал + Shell:** ~200 МБ
   - **Языки:** ~1-2 ГБ
   - **Редакторы:** ~500 МБ - 1 ГБ
   - **Devtools:** ~500 МБ
   - **AI-стек:** ~10-20 ГБ (с моделями)
   - **NVIDIA драйверы:** ~1 ГБ

3. **Устанавливайте по порядку:**
   - Сначала терминал и оболочку
   - Затем языки
   - Потом редакторы и инструменты
   - AI-стек в конце (требует много места)

4. **Проверка после установки:**
   ```bash
   # Проверка версий
   zsh --version
   kitty --version
   python --version
   node --version
   codium --version
   ```

## 🔧 Решение проблем

### Нет сети
Мастер проверит подключение к интернету перед установкой. Если сети нет:
```bash
# Проверьте подключение
ping -c 3 github.com

# Настройте сеть
sudo nmtui
```

### Закончилось место
```bash
# Очистка кэша apt
sudo apt clean
sudo apt autoremove

# Проверка места
df -h
du -sh /var/cache/apt
```

### Ошибка при установке
Если скрипт завершился с ошибкой:
1. Проверьте логи установки
2. Попробуйте установить компонент вручную
3. Убедитесь, что у вас есть права root
