# VibeCode OS Minimal — Состав пакетов

Этот документ описывает пакеты, входящие в Minimal ISO, и их влияние на размер образа.

---

## 📊 Базовая конфигурация (~400-500 МБ)

### Обязательные пакеты

| Категория | Пакеты | Размер | Назначение |
|-----------|--------|--------|------------|
| **Ядро и загрузка** | `linux-image-generic`, `initramfs-tools`, `casper` | ~160 МБ | Загрузка системы, live-сессия |
| **Оболочка** | `zsh`, `tmux` | ~4 МБ | Терминал и мультиплексор |
| **Редакторы** | `nano`, `vim-tiny` | ~800 КБ | Редактирование файлов |
| **Сеть** | `network-manager`, `iputils-ping`, `curl`, `wget`, `ca-certificates` | ~3 МБ | Управление сетью, загрузка файлов, SSL |
| **Мониторинг** | `htop` | ~5 МБ | Монитор процессов |
| **Безопасность** | `sudo` | ~2 МБ | Выполнение от root |
| **Архиваторы** | `unzip` | ~200 КБ | Распаковка ZIP |
| **Виртуализация** | `virtualbox-guest-utils` | ~5 МБ | Интеграция с VirtualBox |

**Итого базовый размер:** ~175 МБ (несжатые пакеты)  
**Размер ISO после сжатия:** ~400-500 МБ

---

## 📦 Опциональные пакеты

### Инструменты разработки (+~50-60 МБ)

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `build-essential` | ~50 МБ | GCC, make, libc-dev |
| `git` | ~10 МБ | Контроль версий |
| `linux-headers-generic` | ~100 МБ | Заголовки ядра (для модулей) |

**Когда нужно:**
- Планируете компилировать программы
- Нужна сборка модулей ядра (NVIDIA, VirtualBox)
- Разработка ПО

**Когда можно убрать:**
- Только готовое окружение
- Нет компиляции

---

### Файловые менеджеры и утилиты (+~5-10 МБ)

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `mc` | ~5 МБ | Midnight Commander |
| `tree` | ~50 КБ | Визуализация дерева файлов |
| `traceroute` | ~100 КБ | Трассировка маршрутов |
| `net-tools` | ~200 КБ | ifconfig, netstat (устарело) |

**Когда нужно:**
- Нужен файловый менеджер в терминале
- Диагностика сети

**Когда можно убрать:**
- Предпочитаете чистую консоль
- Используете современные аналоги (`ip`, `ss`)

---

### Архиваторы (+~2-3 МБ)

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `zip` | ~200 КБ | Создание ZIP архивов |
| `p7zip-full` | ~2 МБ | 7z архивы |

**Когда нужно:**
- Работа с различными форматами архивов

**Когда можно убрать:**
- Только ZIP и стандартные форматы

---

### Системные утилиты (+~2 МБ)

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `software-properties-common` | ~2 МБ | `add-apt-repository` |

**Когда нужно:**
- Добавление PPA репозиториев

**Когда можно убрать:**
- Только официальные репозитории Ubuntu

---

### Красивости (+~500 КБ)

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `neofetch` | ~500 КБ | Информация о системе |

**Когда нужно:**
- Красивый скриншот системы
- Быстрая информация о железе

**Когда можно убрать:**
- Не нужен (есть `uname -a`, `cat /etc/os-release`)

---

## 🔧 Служебные пакеты (НЕ для runtime)

### Только для сборки ISO

| Пакет | Размер | Назначение |
|-------|--------|------------|
| `squashfs-tools` | ~500 КБ | Создание SquashFS (только сборка) |

**Важно:** Эти пакеты нужны только на хосте для сборки ISO, не включайте их в образ!

| Конфигурация | Размер ISO | После установки | Для кого |
|--------------|------------|-----------------|----------|
| **Ultra Minimal** | ~400-500 МБ | ~1.5-2 ГБ | Серверы, контейнеры, CI/CD |

## 🛠️ Как изменить состав

1. Отредактируйте `scripts/base/minimal-packages.sh`
2. Закомментируйте/раскомментируйте нужные секции
3. Пересоберите ISO:

```bash
sudo rm -rf build-minimal/
sudo BUILD_MODE=full ./scripts/build-minimal-iso.sh
```

---

## 📝 Примечания

1. **Ядро** занимает ~150 МБ несжатых данных, но хорошо сжимается в SquashFS
2. **Заголовки ядра** (+100 МБ) нужны только для сборки модулей
3. **VirtualBox guest utils** можно убрать для bare-metal систем
4. **Casper** обязателен для live-сессии, но не нужен после установки
