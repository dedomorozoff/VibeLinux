### Core OS: минимальный набор пакетов для alpha-ISO (черновик)

Этот документ фиксирует ориентировочный список пакетов, которые должны входить в alpha-образ VibeCode OS на уровне базовой системы (без полного dev- и AI-стека).

Список не претендует на абсолютную полноту и будет уточняться по мере тестирования на реальном железе и в виртуальных машинах.

---

### 1. Базовая система и утилиты

**Цель:** рабочая консольная среда, пригодная для администрирования и базовой разработки сразу после установки.

- Базовые инструменты:
  - `htop`
  - `btop`
  - `neofetch` (или аналог, если пакет будет признан устаревшим)
  - `curl`
  - `wget`
  - `unzip`
  - `git`
  - `build-essential`
  - `ca-certificates`
  - `software-properties-common`

- Сеть и обновления:
  - (через мета-пакеты Ubuntu Server / NetworkManager, как часть базовой системы)
  - рабочий `apt` с корректно настроенными зеркалами.

Часть этих пакетов уже устанавливается `scripts/base/base-packages.sh`.

---

### 2. Графическая среда: KDE Plasma

**Цель:** современный, но не перегруженный рабочий стол на базе Plasma.

- Минимальный стек Plasma для alpha:
  - `kde-standard` — базовый сбалансированный набор Plasma + KDE-приложения.
  - `sddm` — дисплей-менеджер по умолчанию для KDE.

Такой набор:
- проще сопровождать, чем полный `kde-full` или kubuntu-мета-пакеты,
- одновременно даёт пользователю полноценный рабочий стол «из коробки».

Установка и конфигурация описана в `scripts/desktop/install-plasma.sh`.

---

### 3. Графические и системные утилиты (минимум)

Для alpha-драфта достаточно того, что приходит вместе с `kde-standard`. При необходимости можно дополнительно зафиксировать:

- файловый менеджер (обычно `dolphin`),
- терминал KDE (`konsole`),
- текстовый редактор (`kate`),
- архиватор (`ark`),
- базовый браузер (как часть статка Kubuntu/Plasma или отдельный выбор позже).

Конкретный перечень приложений будет уточнён по результатам первых тестов образа.

---

### 4. Драйверы и графика

**Открытые драйверы:**
- Xorg/Wayland-стек и драйверы для стандартных GPU по умолчанию (идут вместе с Ubuntu 24.04 и не требуют отдельной фиксации).

**Проприетарные драйверы NVIDIA (alpha-уровень):**

- Базовый пакет драйвера:
  - `nvidia-driver-535`

Для alpha важно:
- иметь возможность установить драйвер **после** инсталляции через скрипт `scripts/drivers/install-nvidia.sh`,
- не усложнять сборку ISO дополнительными зависимостями CUDA.

Расширенный стек (CUDA, `nvidia-cuda-toolkit` и др.) рассматривается как отдельный шаг и будет описан в `docs/DRIVERS-NVIDIA.md`.

---

### 5. Что сознательно не входит в alpha

- Полный мета-пакет `kde-full` и/или «тяжёлые» наборы приложений.
- Большинство dev-инструментов (языки, IDE, Docker и т.п.) — ставятся отдельными скриптами из `scripts/dev/` и описаны в `docs/DEVSTACK.md`.
- AI-стек (Ollama, ComfyUI, Open WebUI и др.) — ставится отдельными скриптами и описан в `docs/AI-STACK.md`.

Цель alpha — надёжная базовая система с Plasma и понятным путём до установки dev/AI-слоёв, а не «максимально нафаршированный» образ.
