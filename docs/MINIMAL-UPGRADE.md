# VibeCode OS — minimal-upgrade.sh

## 📋 Описание

**minimal-upgrade.sh** — интерактивный мастер для доустановки компонентов в VibeCode OS Minimal.

Этот скрипт **включён в состав Minimal ISO** и доступен сразу после установки системы.

## 🚀 Быстрый старт

```bash
# Запуск мастера (требуется sudo)
sudo vibecode-upgrade

# Или напрямую
sudo bash /usr/local/bin/vibecode-upgrade
```

## 📦 Что можно установить

Мастер предлагает 6 категорий компонентов:

### 1. Терминал и оболочка (~200 МБ)
- **Kitty** — GPU-ускоренный терминал
- **Zsh** + **Oh My Zsh** — оболочка с плагинами
- **Starship** — кроссплатформенный промпт
- **CLI-утилиты:**
  - `eza` — современная замена `ls`
  - `bat` — замена `cat` с подсветкой
  - `fd` — быстрая замена `find`
  - `ripgrep (rg)` — быстрый поиск текста
  - `fzf` — нечёткий поиск
  - `zoxide` — умные переходы по каталогам
  - `btop` — монитор ресурсов

### 2. Языки программирования (~1-2 ГБ)
- **Python** (3.11, 3.12) через `pyenv`
- **Node.js** (LTS + latest) через `nvm`
- **Rust** через `rustup`
- **Go**
- **Java 21 LTS** через `SDKMAN!`

### 3. Редакторы и IDE (~500 МБ - 1 ГБ)
- **VSCodium** с расширениями
- **Neovim** + **AstroNvim**
- **Zed**

### 4. Инструменты разработчика (~500 МБ)
- **Git**
- **lazygit**
- **Docker** + **Docker Compose**

### 5. AI-стек (~10-20 ГБ)
- **Ollama** — локальные LLM
- **Open WebUI** — веб-интерфейс
- **ai-chat** — терминальный AI-чат
- **Aider** — AI-парное программирование
- **Python AI-библиотеки** (LangChain, Transformers, LlamaIndex)
- **ComfyUI** — генерация изображений

### 6. Драйверы NVIDIA (~1 ГБ)
- Проприетарные драйверы для видеокарт NVIDIA
- Автоматическое определение оборудования

### Команды

| Команда | Описание |
|---------|----------|
| `1-6` | Установить конкретный компонент |
| `A` | Установить ВСЁ (полная версия) |
| `R` | Перепроверить установленные компоненты |
| `0` или `Q` | Выход |
| `1 2 3` | Установить несколько компонентов (через пробел) |

## 💡 Советы

1. **Проверьте место на диске:**
   ```bash
   df -h
   ```

2. **Проверьте подключение к интернету:**
   ```bash
   ping -c 3 github.com
   ```

3. **Начните с проверки:**
   - Выберите `R` для проверки уже установленных компонентов
   - Уже установленные компоненты будут помечены `[✓]`

4. **Устанавливайте по порядку:**
   - 1 → 2 → 3 → 4 → 5 → 6
   - Это обеспечит правильную установку зависимостей

5. **AI-стек требует много места:**
   - Минимум 10 ГБ
   - Рекомендуется 20+ ГБ

## 🔧 Примеры

### Установить только терминал
```bash
sudo vibecode-upgrade
# Выбрать: 1
```

### Установить терминал и языки
```bash
sudo vibecode-upgrade
# Выбрать: 1 2
```

### Установить всё (Full версия)
```bash
sudo vibecode-upgrade
# Выбрать: A
```

### Проверить установленные компоненты
```bash
sudo vibecode-upgrade
# Выбрать: R
```

## 📝 Требования

- **Права:** Требуется запуск от root (`sudo`)
- **Сеть:** Необходимо подключение к интернету
- **Место на диске:**
  - Минимум: 5 ГБ (без AI-стека)
  - Рекомендуется: 25+ ГБ (с AI-стеком)

## 🐛 Решение проблем

### Нет сети
```bash
# Проверка подключения
ping -c 3 github.com

# Настройка сети (TUI)
sudo nmtui
```

### Закончилось место
```bash
# Очистка кэша apt
sudo apt clean
sudo apt autoremove

# Проверка места
df -h
```

### Ошибка при установке Docker
```bash
# Перезапуск сервиса
sudo systemctl restart docker

# Проверка статуса
sudo systemctl status docker
```

### Ошибка при установке NVIDIA драйверов
```bash
# Проверка наличия GPU
lspci | grep -i nvidia
nvidia-smi

# Если не найдено — драйверы не нужны
```
