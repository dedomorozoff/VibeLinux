# VibeCode OS — Полный список пакетов и программ

Этот документ содержит исчерпывающий список программного обеспечения, входящего в состав VibeCode OS.

---

## 📦 Редакции

### VibeCode OS Full
Полноценная рабочая среда для разработки с GUI, AI-инструментами и полным dev-стеком.

**Размер ISO:** ~3-4 ГБ  
**Размер после установки:** ~15-20 ГБ (с моделями и инструментами)

### VibeCode OS Minimal
Минимальная консольная система для серверов, контейнеров и развёртывания.

**Размер ISO:** ~600-800 МБ (оптимизировано)
**Размер после установки:** ~2-3 ГБ

## 🔧 Требования к ресурсам

### Full версия

**Минимум:**
- RAM: 8 ГБ
- Диск: 25 ГБ
- CPU: 4 ядра

**Рекомендуется:**
- RAM: 16-32 ГБ
- Диск: 50+ ГБ (SSD)
- CPU: 6+ ядер
- GPU: NVIDIA с 8+ ГБ VRAM (для AI)

### Minimal версия

**Минимум:**
- RAM: 1 ГБ
- Диск: 5 ГБ
- CPU: 2 ядра

**Рекомендуется:**
- RAM: 2-4 ГБ
- Диск: 10+ ГБ
- CPU: 4 ядра

---

## 📝 Примечания

1. **AI-модели** загружаются отдельно после установки через `scripts/ai/install-ollama-models.sh`
2. **Проприетарные драйверы NVIDIA** устанавливаются через `scripts/drivers/install-nvidia.sh`
3. **Расширения VSCodium** устанавливаются автоматически при первом запуске
4. **Python AI-библиотеки** устанавливаются в виртуальное окружение `~/.venv-ai`
5. **ComfyUI** требует дополнительной загрузки моделей Stable Diffusion
6. **Minimal → Full:** Используйте мастер доустановки `sudo make upgrade` для доустановки компонентов
