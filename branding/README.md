### Брендинг VibeCode OS

Этот каталог содержит визуальные элементы бренда VibeCode OS.

---

### Структура

- `logos/` — логотипы в различных форматах (SVG, PNG)
- `wallpapers/` — фирменные обои для рабочего стола
- `themes/` — темы оформления (GTK, MATE, иконки)
- `icons/` — иконки приложений и системные иконки

---

### Цветовая палитра

```
Основной фон:    #0B1020 (тёмный синий)
Акцент 1:        #4CC9F0 (неоновый голубой)
Акцент 2:        #7209B7 (фиолетовый)
Успех:           #2EC4B6 (мятный)
Предупреждение:  #FFE066 (мягкий жёлтый)
```

---

### Шрифты

- JetBrains Mono (основной для кода)
- Fira Code (альтернатива)
- Cascadia Code (опциональный)

---

### Использование

Скрипт `scripts/desktop/apply-branding.sh` применяет брендинг к системе во время сборки ISO.
